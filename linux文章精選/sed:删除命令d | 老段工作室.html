sed:删除命令d

使用 sed 的d 指令，你可以删除一些行。请注意这些行只是在输出的时候被删除。和其他命令一样，d命令不会修改源文件的内容。

 

默认情况下，在使用sed前，如果你没有指定任何地址范围 ，它将匹配所有的行。因此，下列的例子将不打印任内容，因为它匹配了 employee.txt 中所有的行并删除了他们。

 

sed 'd' employee.txt

 

在删除时，指定一个位址范围会很有用。下面是一些例子：

只删除第二行：

$ sed '2 d' employee.txt

101,John Doe,CEO

103,Raj Reddy,Sysadmin

104,Anand Ram,Developer

105,Jane Miller,Sales Manager

 

删除1到4行：

$ sed '1,4 d' employee.txt

105,Jane Miller,Sales Manager

 

删除从第二行到最后一行：

$ sed '2,$ d' employee.txt

101,John Doe,CEO

 

只删除奇数行：

$ sed '1~2 d' employee.txt

102,Jason Smith,IT Manager

104,Anand Ram,Developer

 

删除所有包含'Manager'的行：

$ sed '/Manager/ d' employee.txt

101,John Doe,CEO

103,Raj Reddy,Sysadmin

104,Anand Ram,Developer

 

删除第一次包含'Jason'的行到第四行：

$ sed '/Jason/,4 d' employee.txt

101,John Doe,CEO

105,Jane Miller,Sales Manager

如果在前四行没有匹配'Jason'的行，这个命令只删除第四行以后匹配'Jason'的行。

 

删除从第一次匹配'Raj'的行到最后一行：

$ sed '/Raj/,$ d' employee.txt

101,John Doe,CEO

102,Jason Smith,IT Manager

 

删除从第一次匹配'Raj'的行 到匹配'Jane'之间的行：

$ sed '/Raj/,/Jane/ d' employee.txt

101,John Doe,CEO

102,Jason Smith,IT Manager

 

删除匹配'Jason'的行及它下面两行：

$ sed '/Jason/,+2 d' employee.txt

101,John Doe,CEO

105,Jane Miller,Sales Manager

 

一些有用的删除例子
下面的例子在日常的应用中很有用。

从一个文件删除所有空行：

sed '/^$/ d' employee.txt

 

删除所有注释行（假设注释以#开头）：

sed '/^#/ d' employee.txt

 

注意：当你使用多个sed命令的时候，d命令会删除所有匹配的行，其他的命令将不会作用在这些删除的行上。
